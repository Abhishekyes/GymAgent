📱 Mobile Reminder Setup Guide

To run these reminders on your phone automatically, choose one of these simple configurations:

1. Google Calendar (Most Reliable / Multi-device)

  1. Click the Download ICS Calendar button in the header.
  2. Go to calendar.google.com on your computer.
  3. Click Settings (gear icon) ➔ Import & export.
  4. Upload the downloaded muscle_meal_schedule.ics file.
  5. Set up notification sounds/banners on your phone's Calendar App.

2. Apple Reminders / Todoist

  1. Type create Apple Reminder text in the coach console.
  2. Copy the output directly.
  3. Open **Todoist** on desktop, paste the items into the task field (Todoist automatically parses dates like "every day at 6:15 AM").
  4. For **Apple Reminders**, you can create a text list and bulk paste or use the iPhone Shortcuts app to import reminders.

3. iPhone Shortcuts Automation

Create a Shortcut containing a list of your meal times and configure recurring notifications via "Time of Day" Automations. (Use the Copy buttons on each card to paste recipe and ingredients text directly into the reminders).
